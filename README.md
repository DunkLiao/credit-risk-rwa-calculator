# 📊 信用風險標準法適用風險權重計算器 (Basel III / 金管會最新規範)

這是一個專為銀行與金融從業人員設計的**信用風險標準法 (Standardised Approach for Credit Risk) 風險權重與 RWA (風險加權資產) 計算器**。

本專案依據 **巴塞爾資本協定三 (Basel III) 核心改革定案** 及 **台灣金管會最新修正之《銀行自有資本與風險性資產之計算方法說明及表格》** 進行開發，提供高精確度的信用加權資產計量、敏感度分析與資產組合評估。

---

## ✨ 核心功能特色

### 1. 🔍 單筆曝險計量與壓力情境分析
* **完整資產類別支援**：
  * **主權國家與中央銀行**：依外部信用評等（1~6級）自動對應 0% ~ 150% 風險權重。
  * **銀行與證券商**：支援標準評等法，提供一般債權與短期債權（3個月以下）雙重計量。
  * **一般企業**：包含外部信評對照，並支援**合格中小企業 (SME)** 適用 85% 優惠風險權重之計算。
  * **零售債權**：區分「合格零售債權 (75%)」與「其他零售債權 (100%)」。
  * **住宅用不動產 (RRE)**：導入最新 **LTV (貸放成數) 敏感權數矩陣**，支援一般型與收益型（Buy-to-Let，還款實質依賴不動產收益）兩種成數級距（20% ~ 105%）。
  * **商用不動產 (CRE)**：支援基於 LTV 與收益依賴度之風險權重劃分。
  * **土地收購、開發及建築 (ADC 曝險)**：涵蓋住宅區購地 (150%)、商業區購地 (200%)、建築業餘屋 (150%)、工業區閒置土地 (200%) 以及符合特定預售/實質股權條件下適用之 100% 優惠權重。
  * **權益證券 (股權) 與其他資產**：支援上市 (250%)、非上市 (400%)、庫存現金 (0%) 等。
* **表外項目轉換 (CCF)**：可選用信用轉換係數（如 10%, 20%, 50%, 100%），精確計算信用相當曝險額 (EAD)。
* **信用風險抵減 (CRM) 技術**：
  * **綜合法 (Comprehensive Approach)**：支援輸入合格金融擔保品價值，扣除標準折扣率 (Haircut)，並加計貨幣錯配折扣 (FX Haircut, 8%) 計算調整後淨曝險。
  * **簡單法 (Simple Approach)**：提供權重替代法，將擔保/保證覆蓋部分替換為保證人適用權數。
* **動態圖表分析**：即時繪製信用評等變動、LTV 增減對 RWA 及最低資本要求的敏感度圖表。

### 📂 2. 批次大量試算與組合分析
* **Excel 範本匯入**：提供一鍵下載預設欄位之 Excel 批次範本，填寫後即可上傳分析。
* **組合風險儀表板**：自動計算組合總 EAD、總 RWA、加權平均風險權重以及 8% 資本計提要求。
* **圖表視覺化**：直觀呈現各曝險類別的 RWA 分布比例，以及每筆曝險之金額與 RWA 的對比圖。
* **結果匯出**：計算完成後的詳細明細表可一鍵匯出為 Excel 試算表。

### 📜 3. 法規指南與對照表
* 提供 S&P, Moody's, 中華信評對應之信用等級對照表。
* 彙整主權、銀行、企業、零售、RRE（LTV 矩陣）及 ADC 曝險之法規權數速查表。

## 🚀 本機運行指南

### 前置要求
* 系統需安裝 **Python 3.8+**。

### 1. 安裝套件
開啟終端機（CMD 或 PowerShell），切換至本專案目錄並執行：
```bash
pip install -r requirements.txt
```

### 2. 啟動計算器網站
在終端機中執行：
```bash
streamlit run app.py
```
*或是直接**雙擊執行 [run_app.bat](file:///D:/VIbeCoding/risk_analysis/run_app.bat)**，系統會自動在瀏覽器中開啟 `http://localhost:8501`。*

5. 部署完成後，於服務設定頁面產生或綁定公開網域即可上線使用。
